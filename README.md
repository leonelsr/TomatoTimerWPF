# TomatoTimerWPF

Pomodoro timer for windows 7+ (Desktop) showing countdown on taskbar by using OverlayIcon and ProgressBar of the TaskbarManager.
Fork of **Harpseal**'s [TomatoTimerWPF](https://github.com/Harpseal/TomatoTimerWPF)
